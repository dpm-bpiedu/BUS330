-------------------------------------------------------------
SCRIPT - Paste right before </body>

To generate toc, inside () add classes for
1: frontmatter headings
2: chapter heading
3: section headings
4: h2 headings
5: backmatter headings
in quotes, comma seperated.
for example:

generateTOC(".h1", ".co-ct", ".h1", ".h2", ".h2");

To hide media, inside () add classes for 
media elements in quotes, comma seperated.
for example:

hideMedia(".video-center", ".video-right", ".video-right");
-------------------------------------------------------------

<script type="text/javascript">
	generateTOC();	
	hideMedia();
</script>
